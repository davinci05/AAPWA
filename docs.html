Anwesenheit v2.1 - Dokumentation
Nutzungshinweise für die Anwesenheitserfassung

Übersicht

Die Anwesenheits-App dient dazu, auf einfache Weise eine Anwesenheitsmeldung per E-Mail zu versenden. Die Anwendung läuft direkt im Browser und kann auch als Progressive Web App (PWA) auf dem Homescreen installiert werden.

Wichtige Funktionen:

  • Anwesenheit per E-Mail senden
  • Benutzername und Hostname anpassen
  • Zeitauswahl manuell verändern
  • Verschlüsselter Verifizierungscode wird automatisch generiert

Nutzung der App

Grundlegende Bedienung

  1. Geben Sie Ihren Benutzernamen ein oder lassen Sie den voreingestellten Wert
  2. Ändern Sie den Hostname nach Bedarf
  3. Passen Sie die Uhrzeit an (standardmäßig ist die aktuelle Zeit eingestellt)
  4. Klicken Sie auf "Anwesenheit senden", um die E-Mail zu erstellen
  5. Die E-Mail öffnet sich automatisch in Ihrem Standard-E-Mail-Programm
  6. Senden Sie die E-Mail ohne zusätzliche Änderungen ab

Übersicht der angezeigten Informationen

  • Username: Ihr Benutzername
  • Hostname: Name des Computers/Geräts
  • IP-Adresse: Eine generierte IP (19.254.4.x)
  • Zeitstempel: Datum und Uhrzeit

Windows Benutzername und Hostnamen finden

Um Ihren Windows Benutzernamen und Hostnamen zu finden, befolgen Sie diese Schritte:

Option 1: Über die Befehlszeile

  1. Drücken Sie Windows-Taste + R, geben Sie cmd ein und drücken Sie Enter
  2. Im Eingabeaufforderungsfenster geben Sie den folgenden Befehl ein:
echo %USERNAME%@%COMPUTERNAME%

Dies zeigt Ihren Benutzernamen und Computernamen im Format benutzername@computername an.

Option 2: Über die Systemeinstellungen

  1. Klicken Sie auf Start > Einstellungen > System > Über
  2. Hier finden Sie den Gerätenamen (Hostnamen) sowie Informationen zu Ihrem Konto

Outlook Signatur entfernen

Auf dem Computer (Windows/Mac)

  1. Öffnen Sie Outlook
  2. Klicken Sie auf Datei > Optionen > E-Mail > Signaturen
  3. Hier können Sie alle vorhandenen Signaturen bearbeiten oder löschen
  4. Entfernen oder deaktivieren Sie die automatische Signatur für neue Nachrichten

Auf dem iPhone/iPad

  1. Öffnen Sie die Einstellungen App
  2. Scrollen Sie nach unten und tippen Sie auf Mail
  3. Tippen Sie auf Signatur
  4. Löschen Sie den Text "Gesendet von meinem iPhone" oder andere vorhandene Signaturen

In der Outlook App (Android/iOS)

  1. Öffnen Sie die Outlook App
  2. Tippen Sie auf Ihr Profilbild oder Initialen oben links
  3. Tippen Sie auf das Zahnrad-Symbol für Einstellungen
  4. Scrollen Sie nach unten zu E-Mail und tippen Sie auf Signatur
  5. Löschen oder ändern Sie die Signatur nach Bedarf
